如何編譯執行這份程式碼
======================

這份程式碼是用 C 語言寫的。
如果是在 Mac (Unix)、Linux 的環境下，直接在 Commnad Line 當前目錄下 “make run” 這個指令就會自動把編譯好的程式放到 “/bin/main” 路徑下並自動執行。
如果是在Windows作業系統下的話，在 “/bin” 裡就有一個 “main.exe” 檔案，直接執行即可。


關於這份程式碼
==============

執行時會先列出神經元經過訓練後的權重(Weight)和偏移量(Bias)
接著會顯示該神經元的 Learning Rate、經過了幾個世代，以及預測的結果和準確度。

若要參考更多資源，請參考報告書.
